forceSafeArea property

bool forceSafeArea
final

Force using Flutter's SafeArea even when edge-to-edge is active.

Some apps prefer the platform-safe-area semantics provided by the SafeArea widget instead of manually applying MediaQuery padding. Set this to true to always wrap the child with SafeArea when content should avoid system bars.

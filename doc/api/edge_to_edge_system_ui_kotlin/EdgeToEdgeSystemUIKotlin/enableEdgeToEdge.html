Future<bool> enableEdgeToEdge()

Enable edge-to-edge mode on the device (Android).

When enabled, app content will render edge-to-edge (behind the status and navigation bars). The plugin will attempt to set the appropriate window flags and notify the native implementation to update system insets. On success this method updates isEdgeToEdgeEnabled.

Returns true when the native platform successfully changed the edge-to-edge state. Callers should refresh any layout or insets after a successful transition (for example by re-reading getSystemInfo).

Implementation

Future<bool> enableEdgeToEdge() async {
  try {
    final res = await _channel.invokeMethod<bool>('enable');
    final success = res == true;
    if (success) await initialize();
    isEdgeToEdgeEnabled = isEdgeToEdgeEnabled || success;
    return success;
  } on PlatformException catch (e) {
    // ignore: avoid_print
    print('enableEdgeToEdge failed: ${e.message}');
    return false;
  }
}

edge_to_edge_system_ui library

Edge-to-edge system UI utilities for Android.

Provides a small, typed Dart API backed by a Kotlin plugin to query system insets, enable/disable edge-to-edge behavior, and set system bar colors. See the example/ folder for a runnable example.

Version 0.1.0-dev.3 includes:

  • Enhanced API with convenience methods for common use cases
  • Flutter-friendly parameter types (Color objects, Brightness enum)
  • Wrapper widgets for automatic configuration and safe areas
  • Utility functions for Android version mapping
